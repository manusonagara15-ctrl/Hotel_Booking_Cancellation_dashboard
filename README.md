🏨 Hotel Booking Cancellation Dashboard

🔹 Project Description
Developed an interactive Hotel Booking Cancellation Dashboard using Power BI.
Analyzed booking and cancellation trends across multiple years.
Created visualizations to identify factors influencing booking cancellations.
Enabled stakeholders to monitor hotel performance and customer behavior.

🔹 Key Features
📌 Total Bookings and Total Cancellations KPI Cards
📌 Year-wise Booking Analysis (2015–2017)
📌 Guest Type Analysis (Couples, Family, Single)
📌 Desired vs Undesired Room Booking Comparison
📌 City Hotel vs Resort Hotel Booking Distribution
📌 Hotel-wise Cancellation Analysis
📌 Monthly Booking and Cancellation Trends
📌 Interactive Year Filter (Slicer)

🔹 Tools & Technologies
Power BI
Power Query
DAX (Data Analysis Expressions)
Data Modeling
Data Visualization

🔹 Dashboard Insights
Identified total bookings and cancellation rates.
Analyzed cancellation behavior by guest type.
Compared booking trends between City Hotels and Resort Hotels.
Evaluated the impact of room preference on cancellations.
Tracked monthly fluctuations in bookings and cancellations.
Highlighted peak months with the highest cancellation counts.

🔹 Project Objective
Understand hotel booking and cancellation patterns.
Identify key factors contributing to booking cancellations.
Support hotel management in improving occupancy rates.
Enable data-driven decisions to reduce cancellation losses.
